Este conjunto de prompts foi refinado ao longo do projeto com o objetivo de apoiar revisões, aprofundamento de conteúdo e aplicação prática dos conceitos de Clean Code. Eles podem ser utilizados como base para estudos contínuos, organização de conhecimento e desenvolvimento de pensamento crítico


# Para compreender conceitos
-"Explique este conceito de forma simples e didática, como se fosse para um iniciante".

-"Descreva este conceito destacando sua importância no desenvolvimento de software".

-"Explique este conceito com exemplos práticos no dia a dia de um desenvolvedor".

# Estruturar resumos de fácil entendimento

-"Resuma este conteúdo em tópicos claros e objetivos".

-"Organize este conteúdo em formato de checklist para estudo".

-"Transforme este conteúdo em um resumo estruturado para revisão rápida".

# Para conectar teoria com prática:
-"Crie exemplos práticos para este conceito aplicados em código".

-"Explique como aplicar esse conceito em projetos reais".

-"Descreva cenários onde esse conceito é importante no desenvolvimento".

# Para fixação de conteúdo
-"Gere perguntas e respostas para fixação sobre este tema".

-"Crie um pequeno quiz com base neste conteúdo".

-"Liste perguntas que um desenvolvedor iniciante deve saber sobre esse tema".

# Conclusão
Esses prompts foram ajustados ao longo do projeto com base nos resultados obtidos, buscando respostas mais estruturadas, práticas e aplicáveis ao contexto real de desenvolvimento. Eles refletem não apenas o conteúdo estudado, mas também a evolução na forma de utilizar comandos.
