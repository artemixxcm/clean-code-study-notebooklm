

Este material foi criado com o objetivo de reforçar o aprendizado dos conceitos de Clean Code através de perguntas práticas e respostas diretas. Pode ser utilizado para revisão rápida, fixação de conteúdo e preparação para entrevistas técnicas.




# O que é Clean Code?

 Código limpo, legível, fácil de entender e manter ao longo do tempo.



# Qual o principal objetivo do Clean Code?

Tornar o código compreensível e sustentável, facilitando manutenção e colaboração.



# Quais são os principais benefícios do Clean Code?

Melhor legibilidade

Facilidade de manutenção

Redução de bugs

Melhor colaboração em equipe



# Boas Práticas

# O que caracteriza uma boa função?

Pequena, com responsabilidade única e fácil de entender.



#  Por que nomes de variáveis são importantes?

 Porque comunicam intenção e tornam o código mais legível.



# O que é responsabilidade única?

 Cada função ou classe deve ter apenas um motivo para mudar.



# Regra do Escoteiro

# O que diz a Regra do Escoteiro?

 “Deixe o código melhor do que você encontrou.”



# Qual o impacto dessa regra no dia a dia?

 Melhoria contínua do código, evitando acúmulo de problemas técnicos.



# Testes e TDD

# Qual a visão de Robert C. Martin sobre testes?

✔ Testes são parte essencial do desenvolvimento, não apenas uma etapa final.

---

# O que é TDD?

✔ Test Driven Development: escrever testes antes do código.



# Quais são as três leis do TDD?

 1. Escrever um teste antes do código
 2. Escrever apenas o necessário para o teste falhar
 3. Escrever apenas o código necessário para passar



# Qual a importância do TDD?

Melhora a qualidade do código, reduz bugs e aumenta a confiança nas entregas.



# Perguntas para reflexão

# O que acontece quando não aplicamos Clean Code?

Código difícil de manter, maior chance de erros e retrabalho.



# Como Clean Code impacta o trabalho em equipe?

Facilita colaboração, entendimento e evolução do sistema.



# Por que aprender Clean Code desde o início faz diferença?

Evita criação de maus hábitos e acelera o crescimento profissional.
